## setup guide:

### Firstly install FFmpeg(IMPORTANT):

### The easiest way to do it is through the Chocolatey package manager.

### 1) Open Windows PowerShell as administrator.

![image](https://github.com/user-attachments/assets/70e8b7a1-9937-43b4-87ea-57906f602f13)



### 2) Then go to https://chocolatey.org/install and copy this command and run it.

![image](https://github.com/user-attachments/assets/93814eb0-01e5-4776-b42c-7614d228d716)

![image](https://github.com/user-attachments/assets/4e2632a8-c072-4fc9-b763-62ba097c4efd)

### 3) Then install FFmpeg, by running "choco install ffmpeg"

![image](https://github.com/user-attachments/assets/4c7b878a-d188-4168-a8f7-99789d4f0b5c)


### 4) Then Restart your PC.

### 5) And after that make sure FFmpeg is installed correctly by running "ffmpeg --version"

![image](https://github.com/user-attachments/assets/73bb892d-bd23-4c61-a4a3-c14ac32c91a4)

### It should look something like this



### 6) Then download peterplug-blur-app from my repository:

![image](https://github.com/user-attachments/assets/b6577c16-9771-457d-9fe3-d96b6af88999)

![image](https://github.com/user-attachments/assets/8575c3a0-17aa-4df8-8ace-3a722b0bb409)

After that, you should be good to go.
